# Using `mybible-cli` from other applications

* ## [OmegaT scripts to get text from MyBible modules](./OmegaTscripts)

     These are groovy scripts designed to be used in [OmegaT](https://omegat.org). They allow to get text for specified references using a selected module, without calling `mybible-cli` manually.

* ## [OS-specific tools to show Bible text](./scripts)

     These are scripts for MS Windows, macOS and other Unix-like systems to show `mybible-cli`'s gui showing the text of the reference copied to the clipboard.
